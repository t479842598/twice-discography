# MV Proxy Worker

Cloudflare Worker for signed Bilibili MV proxy URLs generated by the backend.

## Required secret

```bash
wrangler secret put MV_PROXY_SIGNING_SECRET
```

Use the same value as the backend `MV_PROXY_SIGNING_SECRET`, then set backend `MV_PROXY_BASE_URL` to the Worker origin or full route, for example `https://mv-proxy.example.workers.dev` or `https://mv-proxy.example.workers.dev/mv-proxy`.

The Worker validates `u`, `r`, `exp`, and `sig`, forwards `Range` requests to Bilibili with a Bilibili `Referer`, and never receives the saved Bilibili Cookie.
